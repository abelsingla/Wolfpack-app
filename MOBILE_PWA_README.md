# Wolfpack Campaign Log for Android and iOS

This is the mobile-friendly PWA version. It runs in the browser and can be installed to the home screen.

## Android

1. Copy the `dist` folder to any static web host, or open it from a local server.
2. Open the app URL in Chrome.
3. Tap the menu button.
4. Tap **Add to Home screen** or **Install app**.

## iOS / iPadOS

1. Copy the `dist` folder to any static web host, or open it from a local server.
2. Open the app URL in Safari.
3. Tap the Share button.
4. Tap **Add to Home Screen**.

## Notes

- Campaign data is stored locally in the browser/app storage.
- Use the export/import buttons in the app to back up or move campaign JSON.
- Native APK and IPA builds require Android Studio/Xcode, signing certificates, and store/developer account setup.
